# Gemini Browser Chat

Минимальный браузерный чат с Gemini API без внешних зависимостей.

## Что внутри

- `server.js` поднимает локальный HTTP-сервер и проксирует запросы в Gemini API.
- `public/` содержит интерфейс чата.
- `.env` хранит серверные настройки и ключ API.

## Запуск

```bash
npm start
```

После запуска откройте:

```text
http://localhost:3000
```

Если `3000` уже занят, сервер автоматически попробует `3001`, `3002` и так далее. Точный адрес он напечатает в терминале.

## Настройки

Скопируйте `.env.example` в `.env`, если хотите использовать другой ключ или другую модель.

Основные переменные:

- `GEMINI_API_KEY`
- `GEMINI_MODEL`
- `GEMINI_FALLBACK_MODEL`
- `PORT`
- `HOST`
- `GEMINI_SYSTEM_PROMPT`

## Важно

Ключ Gemini не должен попадать в клиентский JavaScript или в публичный репозиторий. Если ключ уже публиковался открыто, его лучше перевыпустить и заменить.

По умолчанию приложение использует `gemini-2.5-flash-lite`, а при rate limit может попробовать `GEMINI_FALLBACK_MODEL`.
