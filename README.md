# Antonov AI

Локальный браузерный чат для Gemini API с серверным прокси, выбором моделей, подсветкой кода и кастомным интерфейсом.

## Что умеет

- работает через локальный Node.js сервер без внешних зависимостей
- не светит API-ключ в браузере
- показывает доступные модели Gemini в селекторе
- умеет переживать rate limit и использовать fallback-модель
- рендерит code blocks и inline code с подсветкой
- хранит историю диалога и выбранную модель в `localStorage`

## Стек

- `Node.js` HTTP server
- нативный `fetch` для Gemini API
- чистый `HTML`, `CSS`, `JavaScript`

## Запуск

```bash
npm start
```

После запуска откройте адрес, который сервер напечатает в терминале. Обычно это:

```text
http://localhost:3000
```

Если порт `3000` занят, сервер сам попробует следующий свободный порт.

## Настройка

1. Откройте `.env`
2. Укажите свой ключ Gemini
3. При необходимости поменяйте стартовую модель и промпт

Пример переменных:

```env
GEMINI_API_KEY=your_key_here
GEMINI_MODEL=gemini-2.5-flash-lite
GEMINI_FALLBACK_MODEL=gemini-flash-lite-latest
PORT=3000
HOST=0.0.0.0
GEMINI_SYSTEM_PROMPT="You are Antonov AI."
```

## Основные файлы

- `server.js` — локальный сервер, прокси к Gemini API, список моделей, rate limit handling
- `public/index.html` — структура интерфейса
- `public/styles.css` — визуальный стиль и анимации
- `public/app.js` — клиентская логика, рендер сообщений, выбор модели, подсветка кода

## Безопасность

- не кладите `GEMINI_API_KEY` в клиентский код
- не коммитьте `.env` в публичный репозиторий
- если ключ уже публиковался, лучше перевыпустить его и заменить
